50-30-20 Budget Helper

This project will help you budget and manage your money using the popular "50-30-20" rule.

What is the 50-30-20 rule?

The 50-30-20 rule is a popular financing rule that divides your expenses in three separate categories. Needs, Wants, and Savings, respectively. An in-depth article can be found below.

How to use?

Before you start using this program, you need to know what some terms mean.

  • Budgeting: Organization of money, and spendeture.
  • Income: The money you receive via work, allowance, etc.
  • Expenses: The amount of money you spend.
